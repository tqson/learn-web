1 Flexbox Layout
  • Mô hình layout một chiều
  • Linh hoạt trong căn chỉnh
  • Dễ dàng responsive
display: flex;
justify-content: center;
2 Grid Layout
  • Hệ thống layout 2 chiều
  • Quản lý layout phức tạp
  • Kiểm soát chính xác
display: grid;
grid-template-columns: 1fr 1fr;
3 Media Queries
  • Điều chỉnh layout theo kích thước màn hình
  • Breakpoints: mobile, tablet, desktop
  • Responsive design
@media (max-width: 768px) {
  .container { width: 100%; }
}
4 CSS Animation
  • Tạo chuyển động mượt mà
  • Keyframes định nghĩa các trạng thái
  • Timing functions điều chỉnh tốc độ
@keyframes slide {
  from { left: 0; }
  to { left: 100px; }
}
5 CSS Transition
  • Hiệu ứng chuyển đổi mượt mà
  • Áp dụng cho hover, active states
  • Điều chỉnh thời gian và kiểu chuyển đổi
transition: all 0.3s ease;
6 Position
  • Relative: vị trí tương đối
  • Absolute: vị trí tuyệt đối
  • Fixed: cố định trên viewport
  • Sticky: bám dính khi scroll
position: absolute;
top: 0; left: 0;
7 Box Model
  • Content: nội dung
  • Padding: đệm trong
  • Border: viền
  • Margin: khoảng cách ngoài
padding: 10px;
margin: 15px;
border: 1px solid;
8 Pseudo Classes
  • :hover - di chuột qua
  • :active - khi click
  • :focus - khi focus
  • :nth-child() - chọn phần tử theo index
.button:hover {
  background: blue;
}
9 Transform
  • Scale: phóng to, thu nhỏ
  • Rotate: xoay
  • Translate: di chuyển
  • Skew: nghiêng
transform: scale(1.2)
  rotate(45deg);
10 Flexbox Properties
  • flex-direction: hướng chính
  • justify-content: căn chỉnh trục chính
  • align-items: căn chỉnh trục phụ
  • flex-wrap: xuống dòng
display: flex;
justify-content: space-between;
align-items: center;
